# Лабораторна робота №1 — Варіант 2 (v2)

## Завдання 2: Виведення тексту

Виведіть наступний текст з форматуванням (жирний, курсив, відступи):

```
Програмування — це мистецтво,
Де логіка зустрічає творчість,
Де код оживає на екрані,
    І створює
        нові
            можливості!
```

- Слово "мистецтво" — **жирним**
- Слово "творчість" — *курсивом*
- Останні 3 рядки — з прогресивним відступом

---

## Завдання 3: Конвертер валют

Напишіть програму, що переводить **фунти стерлінгів в гривні**.

**Вхідні дані:**
- Сума: **100 фунтів**
- Курс: **1 GBP = 52.80 грн**

**Очікуваний вивід:**
```
100 фунтів = 5280 грн
```

---

## Завдання 4: День тижня (if-else)

Напишіть програму, яка визначає **тип дня** за номером дня тижня (1-7):

| День | Тип |
|------|-----|
| 1 (Пн) - 5 (Пт) | Робочий день |
| 6 (Сб) | Вихідний (короткий) |
| 7 (Нд) | Вихідний (повний) |

**Вхідні дані:** день = **6**

---

## Завдання 5: Оцінка (switch)

Напишіть програму, яка переводить **числову оцінку** (1-5) в текстову:

| Оцінка | Текст |
|--------|-------|
| 5 | Відмінно |
| 4 | Добре |
| 3 | Задовільно |
| 2 | Незадовільно |
| 1 | Погано |

Використайте конструкцію `switch`.

**Вхідні дані:** оцінка = **4**

## Завдання 6: П'ятизначне число

Дано випадкове **п'ятизначне** число (10000-99999). Знайдіть:

1. Суму його цифр
2. Середню цифру (третю)
3. Число в зворотному порядку
4. Найменше можливе число з цих цифр (перша цифра не може бути 0)

## Завдання 7.1: Градієнтна таблиця

Напишіть функцію, яка виводить таблицю n×n, де колір кожної комірки залежить від її позиції (градієнт від червоного до синього).

- Розмір: **10×10**

---

## Завдання 7.2: Випадкові трикутники

Напишіть функцію, яка виводить на **темно-зеленому тлі** `n` **білих трикутників** випадкового розміру в випадковій позиції.

- Кількість: **8 трикутників**
- Розмір: від 30px до 100px

*Підказка: використовуйте CSS border для створення трикутників*
