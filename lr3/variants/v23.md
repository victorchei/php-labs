# Варіант 23 — Лабораторна робота №3

> Варіант: 23
> Лаба: LR3 — ООП в PHP

## Завдання

### Завдання 1: Створення класів та об'єктів

- Клас `Project` із властивостями: `name` (string), `status` (string), `budget` (float)
- Створити 3 об'єкти з довільними значеннями:

| # | name | status | budget |
|---|------|--------|--------|
| 1 | E-commerce | В розробці | 150000.00 |
| 2 | CRM | Тестування | 85000.00 |
| 3 | Mobile App | Планування | 200000.00 |

### Завдання 2: Методи об'єкта

- Метод `getInfo()` — повертає рядок: `"Проєкт: {name}, Статус: {status}, Бюджет: {budget} грн"`
- Викликати для кожного об'єкта

**Очікуваний результат:**
```
$project1: Проєкт: E-commerce, Статус: В розробці, Бюджет: 150000 грн
$project2: Проєкт: CRM, Статус: Тестування, Бюджет: 85000 грн
$project3: Проєкт: Mobile App, Статус: Планування, Бюджет: 200000 грн
```

### Завдання 3: Конструктор

- Конструктор задає початкові значення `name`, `status`, `budget`
- Перестворити 3 об'єкти через конструктор (ті самі дані)

### Завдання 4: Клонування

- Метод `__clone()` — при копіюванні задає значення за замовчанням:
  - `name = "Новий проєкт"`, `status = "Чернетка"`, `budget = 0.0`
- Створити 4-й об'єкт через `clone` (від $project3)

**Очікуваний результат:**
```
$project3: Проєкт: Mobile App, Статус: Планування, Бюджет: 200000 грн
$project4: Проєкт: Новий проєкт, Статус: Чернетка, Бюджет: 0 грн
```
