# Варіант 29 — Лабораторна робота №3

> Варіант: 29
> Лаба: LR3 — ООП в PHP

## Завдання

### Завдання 1: Створення класів та об'єктів

- Клас `Flower` із властивостями: `name` (string), `color` (string), `price` (float)
- Створити 3 об'єкти з довільними значеннями:

| # | name | color | price |
|---|------|-------|-------|
| 1 | Троянда | Червоний | 85.00 |
| 2 | Тюльпан | Жовтий | 45.00 |
| 3 | Хризантема | Білий | 60.00 |

### Завдання 2: Методи об'єкта

- Метод `getInfo()` — повертає рядок: `"Квітка: {name}, Колір: {color}, Ціна: {price} грн"`
- Викликати для кожного об'єкта

**Очікуваний результат:**
```
$flower1: Квітка: Троянда, Колір: Червоний, Ціна: 85 грн
$flower2: Квітка: Тюльпан, Колір: Жовтий, Ціна: 45 грн
$flower3: Квітка: Хризантема, Колір: Білий, Ціна: 60 грн
```

### Завдання 3: Конструктор

- Конструктор задає початкові значення `name`, `color`, `price`
- Перестворити 3 об'єкти через конструктор (ті самі дані)

### Завдання 4: Клонування

- Метод `__clone()` — при копіюванні задає значення за замовчанням:
  - `name = "Квітка"`, `color = ""`, `price = 0.0`
- Створити 4-й об'єкт через `clone` (від $flower3)

**Очікуваний результат:**
```
$flower3: Квітка: Хризантема, Колір: Білий, Ціна: 60 грн
$flower4: Квітка: Квітка, Колір: , Ціна: 0 грн
```
