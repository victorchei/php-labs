# Варіант 16 — Лабораторна робота №3

> Варіант: 16
> Лаба: LR3 — ООП в PHP

## Завдання

### Завдання 1: Створення класів та об'єктів

- Клас `Game` із властивостями: `title` (string), `genre` (string), `rating` (float)
- Створити 3 об'єкти з довільними значеннями:

| # | title | genre | rating |
|---|-------|-------|--------|
| 1 | The Witcher 3 | RPG | 9.5 |
| 2 | Cyberpunk 2077 | RPG | 8.6 |
| 3 | S.T.A.L.K.E.R. 2 | Shooter | 8.2 |

### Завдання 2: Методи об'єкта

- Метод `getInfo()` — повертає рядок: `"Гра: {title}, Жанр: {genre}, Рейтинг: {rating}"`
- Викликати для кожного об'єкта

**Очікуваний результат:**
```
$game1: Гра: The Witcher 3, Жанр: RPG, Рейтинг: 9.5
$game2: Гра: Cyberpunk 2077, Жанр: RPG, Рейтинг: 8.6
$game3: Гра: S.T.A.L.K.E.R. 2, Жанр: Shooter, Рейтинг: 8.2
```

### Завдання 3: Конструктор

- Конструктор задає початкові значення `title`, `genre`, `rating`
- Перестворити 3 об'єкти через конструктор (ті самі дані)

### Завдання 4: Клонування

- Метод `__clone()` — при копіюванні задає значення за замовчанням:
  - `title = "Нова гра"`, `genre = ""`, `rating = 0.0`
- Створити 4-й об'єкт через `clone` (від $game3)

**Очікуваний результат:**
```
$game3: Гра: S.T.A.L.K.E.R. 2, Жанр: Shooter, Рейтинг: 8.2
$game4: Гра: Нова гра, Жанр: , Рейтинг: 0
```

## Примітки

- Структура файлів має відповідати demo
