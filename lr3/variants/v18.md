# Варіант 18 — Лабораторна робота №3

> Варіант: 18
> Лаба: LR3 — ООП в PHP

## Завдання

### Завдання 1: Створення класів та об'єктів

- Клас `Apartment` із властивостями: `address` (string), `area` (float), `rooms` (int)
- Створити 3 об'єкти з довільними значеннями:

| # | address | area | rooms |
|---|---------|------|-------|
| 1 | вул. Хрещатик 15 | 75.5 | 3 |
| 2 | вул. Грушевського 8 | 120.0 | 4 |
| 3 | пр. Шевченка 42 | 45.0 | 1 |

### Завдання 2: Методи об'єкта

- Метод `getInfo()` — повертає рядок: `"Квартира: {address}, Площа: {area} м², Кімнат: {rooms}"`
- Викликати для кожного об'єкта

**Очікуваний результат:**
```
$apartment1: Квартира: вул. Хрещатик 15, Площа: 75.5 м², Кімнат: 3
$apartment2: Квартира: вул. Грушевського 8, Площа: 120 м², Кімнат: 4
$apartment3: Квартира: пр. Шевченка 42, Площа: 45 м², Кімнат: 1
```

### Завдання 3: Конструктор

- Конструктор задає початкові значення `address`, `area`, `rooms`
- Перестворити 3 об'єкти через конструктор (ті самі дані)

### Завдання 4: Клонування

- Метод `__clone()` — при копіюванні задає значення за замовчанням:
  - `address = ""`, `area = 0.0`, `rooms = 0`
- Створити 4-й об'єкт через `clone` (від $apartment3)

**Очікуваний результат:**
```
$apartment3: Квартира: пр. Шевченка 42, Площа: 45 м², Кімнат: 1
$apartment4: Квартира: , Площа: 0 м², Кімнат: 0
```
