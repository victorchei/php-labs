# Варіант 8 — Лабораторна робота №3

> Варіант: 8
> Лаба: LR3 — ООП в PHP

## Завдання

### Завдання 1: Створення класів та об'єктів

- Клас `Pilot` із властивостями: `name` (string), `airline` (string), `flightHours` (int)
- Створити 3 об'єкти з довільними значеннями:

| # | name | airline | flightHours |
|---|------|---------|-------------|
| 1 | Павло Марченко | МАУ | 5200 |
| 2 | Вікторія Тимошенко | SkyUp | 3100 |
| 3 | Григорій Клименко | Windrose | 7800 |

### Завдання 2: Методи об'єкта

- Метод `getInfo()` — повертає рядок: `"Пілот: {name}, Авіакомпанія: {airline}, Наліт: {flightHours} год"`
- Викликати для кожного об'єкта

**Очікуваний результат:**
```
$pilot1: Пілот: Павло Марченко, Авіакомпанія: МАУ, Наліт: 5200 год
$pilot2: Пілот: Вікторія Тимошенко, Авіакомпанія: SkyUp, Наліт: 3100 год
$pilot3: Пілот: Григорій Клименко, Авіакомпанія: Windrose, Наліт: 7800 год
```

### Завдання 3: Конструктор

- Конструктор задає початкові значення `name`, `airline`, `flightHours`
- Перестворити 3 об'єкти через конструктор (ті самі дані)

### Завдання 4: Клонування

- Метод `__clone()` — при копіюванні задає значення за замовчанням:
  - `name = "Пілот"`, `airline = ""`, `flightHours = 0`
- Створити 4-й об'єкт через `clone` (від $pilot3)

**Очікуваний результат:**
```
$pilot3: Пілот: Григорій Клименко, Авіакомпанія: Windrose, Наліт: 7800 год
$pilot4: Пілот: Пілот, Авіакомпанія: , Наліт: 0 год
```
