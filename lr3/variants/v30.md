# Варіант 30 — Лабораторна робота №3

> Варіант: 30
> Лаба: LR3 — ООП в PHP

## Завдання

### Завдання 1: Створення класів та об'єктів

- Клас `Product` із властивостями: `name` (string), `price` (float), `category` (string)
- Створити 3 об'єкти з довільними значеннями:

| # | name | price | category |
|---|------|-------|----------|
| 1 | Ноутбук ASUS | 32500.00 | Електроніка |
| 2 | Кросівки Nike | 4200.00 | Взуття |
| 3 | Рюкзак Osprey | 6800.00 | Аксесуари |

### Завдання 2: Методи об'єкта

- Метод `getInfo()` — повертає рядок: `"Товар: {name}, Ціна: {price} грн, Категорія: {category}"`
- Викликати для кожного об'єкта

**Очікуваний результат:**
```
$product1: Товар: Ноутбук ASUS, Ціна: 32500 грн, Категорія: Електроніка
$product2: Товар: Кросівки Nike, Ціна: 4200 грн, Категорія: Взуття
$product3: Товар: Рюкзак Osprey, Ціна: 6800 грн, Категорія: Аксесуари
```

### Завдання 3: Конструктор

- Конструктор задає початкові значення `name`, `price`, `category`
- Перестворити 3 об'єкти через конструктор (ті самі дані)

### Завдання 4: Клонування

- Метод `__clone()` — при копіюванні задає значення за замовчанням:
  - `name = "Новий товар"`, `price = 0.0`, `category = "Без категорії"`
- Створити 4-й об'єкт через `clone` (від $product3)

**Очікуваний результат:**
```
$product3: Товар: Рюкзак Osprey, Ціна: 6800 грн, Категорія: Аксесуари
$product4: Товар: Новий товар, Ціна: 0 грн, Категорія: Без категорії
```
