# Варіант 2 — Лабораторна робота №3

> Варіант: 2
> Лаба: LR3 — ООП в PHP

## Завдання

### Завдання 1: Створення класів та об'єктів

- Клас `Teacher` із властивостями: `name` (string), `subject` (string), `experience` (int)
- Створити 3 об'єкти з довільними значеннями:

| # | name | subject | experience |
|---|------|---------|------------|
| 1 | Наталія Шевченко | Математика | 15 |
| 2 | Олег Бондаренко | Фізика | 8 |
| 3 | Ірина Мельник | Хімія | 12 |

### Завдання 2: Методи об'єкта

- Метод `getInfo()` — повертає рядок: `"Викладач: {name}, Предмет: {subject}, Досвід: {experience} р."`
- Викликати для кожного об'єкта

**Очікуваний результат:**
```
$teacher1: Викладач: Наталія Шевченко, Предмет: Математика, Досвід: 15 р.
$teacher2: Викладач: Олег Бондаренко, Предмет: Фізика, Досвід: 8 р.
$teacher3: Викладач: Ірина Мельник, Предмет: Хімія, Досвід: 12 р.
```

### Завдання 3: Конструктор

- Конструктор задає початкові значення `name`, `subject`, `experience`
- Перестворити 3 об'єкти через конструктор (ті самі дані)

### Завдання 4: Клонування

- Метод `__clone()` — при копіюванні задає значення за замовчанням:
  - `name = "Викладач"`, `subject = ""`, `experience = 0`
- Створити 4-й об'єкт через `clone` (від $teacher3)

**Очікуваний результат:**
```
$teacher3: Викладач: Ірина Мельник, Предмет: Хімія, Досвід: 12 р.
$teacher4: Викладач: Викладач, Предмет: , Досвід: 0 р.
```
