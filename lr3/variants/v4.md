# Варіант 4 — Лабораторна робота №3

> Варіант: 4
> Лаба: LR3 — ООП в PHP

## Завдання

### Завдання 1: Створення класів та об'єктів

- Клас `Athlete` із властивостями: `name` (string), `sport` (string), `rating` (int)
- Створити 3 об'єкти з довільними значеннями:

| # | name | sport | rating |
|---|------|-------|--------|
| 1 | Дарія Білоус | Плавання | 1520 |
| 2 | Тарас Гнатюк | Легка атлетика | 1340 |
| 3 | Софія Приходько | Гімнастика | 1680 |

### Завдання 2: Методи об'єкта

- Метод `getInfo()` — повертає рядок: `"Спортсмен: {name}, Вид спорту: {sport}, Рейтинг: {rating}"`
- Викликати для кожного об'єкта

**Очікуваний результат:**
```
$athlete1: Спортсмен: Дарія Білоус, Вид спорту: Плавання, Рейтинг: 1520
$athlete2: Спортсмен: Тарас Гнатюк, Вид спорту: Легка атлетика, Рейтинг: 1340
$athlete3: Спортсмен: Софія Приходько, Вид спорту: Гімнастика, Рейтинг: 1680
```

### Завдання 3: Конструктор

- Конструктор задає початкові значення `name`, `sport`, `rating`
- Перестворити 3 об'єкти через конструктор (ті самі дані)

### Завдання 4: Клонування

- Метод `__clone()` — при копіюванні задає значення за замовчанням:
  - `name = "Спортсмен"`, `sport = ""`, `rating = 0`
- Створити 4-й об'єкт через `clone` (від $athlete3)

**Очікуваний результат:**
```
$athlete3: Спортсмен: Софія Приходько, Вид спорту: Гімнастика, Рейтинг: 1680
$athlete4: Спортсмен: Спортсмен, Вид спорту: , Рейтинг: 0
```
