# Варіант 15 — Лабораторна робота №3

> Варіант: 15
> Лаба: LR3 — ООП в PHP

## Завдання

### Завдання 1: Створення класів та об'єктів

- Клас `Song` із властивостями: `title` (string), `artist` (string), `duration` (int)
- Створити 3 об'єкти з довільними значеннями:

| # | title | artist | duration |
|---|-------|--------|----------|
| 1 | Червона рута | Софія Ротару | 215 |
| 2 | Обійми | Океан Ельзи | 264 |
| 3 | Плакала | KAZKA | 198 |

### Завдання 2: Методи об'єкта

- Метод `getInfo()` — повертає рядок: `"Пісня: {title}, Виконавець: {artist}, Тривалість: {duration} сек"`
- Викликати для кожного об'єкта

**Очікуваний результат:**
```
$song1: Пісня: Червона рута, Виконавець: Софія Ротару, Тривалість: 215 сек
$song2: Пісня: Обійми, Виконавець: Океан Ельзи, Тривалість: 264 сек
$song3: Пісня: Плакала, Виконавець: KAZKA, Тривалість: 198 сек
```

### Завдання 3: Конструктор

- Конструктор задає початкові значення `title`, `artist`, `duration`
- Перестворити 3 об'єкти через конструктор (ті самі дані)

### Завдання 4: Клонування

- Метод `__clone()` — при копіюванні задає значення за замовчанням:
  - `title = "Без назви"`, `artist = ""`, `duration = 0`
- Створити 4-й об'єкт через `clone` (від $song3)

**Очікуваний результат:**
```
$song3: Пісня: Плакала, Виконавець: KAZKA, Тривалість: 198 сек
$song4: Пісня: Без назви, Виконавець: , Тривалість: 0 сек
```
