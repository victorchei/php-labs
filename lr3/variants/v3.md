# Варіант 3 — Лабораторна робота №3

> Варіант: 3
> Лаба: LR3 — ООП в PHP

## Завдання

### Завдання 1: Створення класів та об'єктів

- Клас `Doctor` із властивостями: `name` (string), `specialization` (string), `licenseNumber` (string)
- Створити 3 об'єкти з довільними значеннями:

| # | name | specialization | licenseNumber |
|---|------|----------------|---------------|
| 1 | Андрій Кравченко | Кардіолог | LIC-4521 |
| 2 | Людмила Савченко | Терапевт | LIC-7834 |
| 3 | Максим Олійник | Хірург | LIC-2190 |

### Завдання 2: Методи об'єкта

- Метод `getInfo()` — повертає рядок: `"Лікар: {name}, Спеціалізація: {specialization}, Ліцензія: {licenseNumber}"`
- Викликати для кожного об'єкта

**Очікуваний результат:**
```
$doctor1: Лікар: Андрій Кравченко, Спеціалізація: Кардіолог, Ліцензія: LIC-4521
$doctor2: Лікар: Людмила Савченко, Спеціалізація: Терапевт, Ліцензія: LIC-7834
$doctor3: Лікар: Максим Олійник, Спеціалізація: Хірург, Ліцензія: LIC-2190
```

### Завдання 3: Конструктор

- Конструктор задає початкові значення `name`, `specialization`, `licenseNumber`
- Перестворити 3 об'єкти через конструктор (ті самі дані)

### Завдання 4: Клонування

- Метод `__clone()` — при копіюванні задає значення за замовчанням:
  - `name = "Лікар"`, `specialization = ""`, `licenseNumber = ""`
- Створити 4-й об'єкт через `clone` (від $doctor3)

**Очікуваний результат:**
```
$doctor3: Лікар: Максим Олійник, Спеціалізація: Хірург, Ліцензія: LIC-2190
$doctor4: Лікар: Лікар, Спеціалізація: , Ліцензія:
```
