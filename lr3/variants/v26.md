# Варіант 26 — Лабораторна робота №3

> Варіант: 26
> Лаба: LR3 — ООП в PHP

## Завдання

### Завдання 1: Створення класів та об'єктів

- Клас `Medicine` із властивостями: `name` (string), `manufacturer` (string), `price` (float)
- Створити 3 об'єкти з довільними значеннями:

| # | name | manufacturer | price |
|---|------|--------------|-------|
| 1 | Панадол | GlaxoSmithKline | 185.50 |
| 2 | Ібупрофен | Дарниця | 95.00 |
| 3 | Амоксицилін | Київмедпрепарат | 142.30 |

### Завдання 2: Методи об'єкта

- Метод `getInfo()` — повертає рядок: `"Ліки: {name}, Виробник: {manufacturer}, Ціна: {price} грн"`
- Викликати для кожного об'єкта

**Очікуваний результат:**
```
$medicine1: Ліки: Панадол, Виробник: GlaxoSmithKline, Ціна: 185.5 грн
$medicine2: Ліки: Ібупрофен, Виробник: Дарниця, Ціна: 95 грн
$medicine3: Ліки: Амоксицилін, Виробник: Київмедпрепарат, Ціна: 142.3 грн
```

### Завдання 3: Конструктор

- Конструктор задає початкові значення `name`, `manufacturer`, `price`
- Перестворити 3 об'єкти через конструктор (ті самі дані)

### Завдання 4: Клонування

- Метод `__clone()` — при копіюванні задає значення за замовчанням:
  - `name = "Невідомі ліки"`, `manufacturer = ""`, `price = 0.0`
- Створити 4-й об'єкт через `clone` (від $medicine3)

**Очікуваний результат:**
```
$medicine3: Ліки: Амоксицилін, Виробник: Київмедпрепарат, Ціна: 142.3 грн
$medicine4: Ліки: Невідомі ліки, Виробник: , Ціна: 0 грн
```
