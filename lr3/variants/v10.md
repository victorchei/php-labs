# Варіант 10 — Лабораторна робота №3

> Варіант: 10
> Лаба: LR3 — ООП в PHP

## Завдання

### Завдання 1: Створення класів та об'єктів

- Клас `Writer` із властивостями: `name` (string), `genre` (string), `booksCount` (int)
- Створити 3 об'єкти з довільними значеннями:

| # | name | genre | booksCount |
|---|------|-------|------------|
| 1 | Сергій Франко | Детектив | 14 |
| 2 | Алла Довженко | Романтика | 9 |
| 3 | Михайло Стефаник | Фантастика | 22 |

### Завдання 2: Методи об'єкта

- Метод `getInfo()` — повертає рядок: `"Письменник: {name}, Жанр: {genre}, Книг: {booksCount}"`
- Викликати для кожного об'єкта

**Очікуваний результат:**
```
$writer1: Письменник: Сергій Франко, Жанр: Детектив, Книг: 14
$writer2: Письменник: Алла Довженко, Жанр: Романтика, Книг: 9
$writer3: Письменник: Михайло Стефаник, Жанр: Фантастика, Книг: 22
```

### Завдання 3: Конструктор

- Конструктор задає початкові значення `name`, `genre`, `booksCount`
- Перестворити 3 об'єкти через конструктор (ті самі дані)

### Завдання 4: Клонування

- Метод `__clone()` — при копіюванні задає значення за замовчанням:
  - `name = "Письменник"`, `genre = ""`, `booksCount = 0`
- Створити 4-й об'єкт через `clone` (від $writer3)

**Очікуваний результат:**
```
$writer3: Письменник: Михайло Стефаник, Жанр: Фантастика, Книг: 22
$writer4: Письменник: Письменник, Жанр: , Книг: 0
```
