# Варіант 17 — Лабораторна робота №3

> Варіант: 17
> Лаба: LR3 — ООП в PHP

## Завдання

### Завдання 1: Створення класів та об'єктів

- Клас `Laptop` із властивостями: `brand` (string), `processor` (string), `ram` (int)
- Створити 3 об'єкти з довільними значеннями:

| # | brand | processor | ram |
|---|-------|-----------|-----|
| 1 | ASUS | Intel i7 | 16 |
| 2 | Lenovo | AMD Ryzen 9 | 32 |
| 3 | HP | Intel i5 | 8 |

### Завдання 2: Методи об'єкта

- Метод `getInfo()` — повертає рядок: `"Ноутбук: {brand}, Процесор: {processor}, RAM: {ram} ГБ"`
- Викликати для кожного об'єкта

**Очікуваний результат:**
```
$laptop1: Ноутбук: ASUS, Процесор: Intel i7, RAM: 16 ГБ
$laptop2: Ноутбук: Lenovo, Процесор: AMD Ryzen 9, RAM: 32 ГБ
$laptop3: Ноутбук: HP, Процесор: Intel i5, RAM: 8 ГБ
```

### Завдання 3: Конструктор

- Конструктор задає початкові значення `brand`, `processor`, `ram`
- Перестворити 3 об'єкти через конструктор (ті самі дані)

### Завдання 4: Клонування

- Метод `__clone()` — при копіюванні задає значення за замовчанням:
  - `brand = ""`, `processor = ""`, `ram = 0`
- Створити 4-й об'єкт через `clone` (від $laptop3)

**Очікуваний результат:**
```
$laptop3: Ноутбук: HP, Процесор: Intel i5, RAM: 8 ГБ
$laptop4: Ноутбук: , Процесор: , RAM: 0 ГБ
```
