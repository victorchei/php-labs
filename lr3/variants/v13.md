# Варіант 13 — Лабораторна робота №3

> Варіант: 13
> Лаба: LR3 — ООП в PHP

## Завдання

### Завдання 1: Створення класів та об'єктів

- Клас `Phone` із властивостями: `brand` (string), `model` (string), `storage` (int)
- Створити 3 об'єкти з довільними значеннями:

| # | brand | model | storage |
|---|-------|-------|---------|
| 1 | Apple | iPhone 15 | 256 |
| 2 | Samsung | Galaxy S24 | 128 |
| 3 | Xiaomi | 14 Pro | 512 |

### Завдання 2: Методи об'єкта

- Метод `getInfo()` — повертає рядок: `"Телефон: {brand} {model}, Пам'ять: {storage} ГБ"`
- Викликати для кожного об'єкта

**Очікуваний результат:**
```
$phone1: Телефон: Apple iPhone 15, Пам'ять: 256 ГБ
$phone2: Телефон: Samsung Galaxy S24, Пам'ять: 128 ГБ
$phone3: Телефон: Xiaomi 14 Pro, Пам'ять: 512 ГБ
```

### Завдання 3: Конструктор

- Конструктор задає початкові значення `brand`, `model`, `storage`
- Перестворити 3 об'єкти через конструктор (ті самі дані)

### Завдання 4: Клонування

- Метод `__clone()` — при копіюванні задає значення за замовчанням:
  - `brand = ""`, `model = ""`, `storage = 0`
- Створити 4-й об'єкт через `clone` (від $phone3)

**Очікуваний результат:**
```
$phone3: Телефон: Xiaomi 14 Pro, Пам'ять: 512 ГБ
$phone4: Телефон:  , Пам'ять: 0 ГБ
```
