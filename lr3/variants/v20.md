# Варіант 20 — Лабораторна робота №3

> Варіант: 20
> Лаба: LR3 — ООП в PHP

## Завдання

### Завдання 1: Створення класів та об'єктів

- Клас `Planet` із властивостями: `name` (string), `diameter` (float), `moons` (int)
- Створити 3 об'єкти з довільними значеннями:

| # | name | diameter | moons |
|---|------|----------|-------|
| 1 | Земля | 12742.0 | 1 |
| 2 | Марс | 6779.0 | 2 |
| 3 | Юпітер | 139820.0 | 95 |

### Завдання 2: Методи об'єкта

- Метод `getInfo()` — повертає рядок: `"Планета: {name}, Діаметр: {diameter} км, Супутників: {moons}"`
- Викликати для кожного об'єкта

**Очікуваний результат:**
```
$planet1: Планета: Земля, Діаметр: 12742 км, Супутників: 1
$planet2: Планета: Марс, Діаметр: 6779 км, Супутників: 2
$planet3: Планета: Юпітер, Діаметр: 139820 км, Супутників: 95
```

### Завдання 3: Конструктор

- Конструктор задає початкові значення `name`, `diameter`, `moons`
- Перестворити 3 об'єкти через конструктор (ті самі дані)

### Завдання 4: Клонування

- Метод `__clone()` — при копіюванні задає значення за замовчанням:
  - `name = "Невідома"`, `diameter = 0.0`, `moons = 0`
- Створити 4-й об'єкт через `clone` (від $planet3)

**Очікуваний результат:**
```
$planet3: Планета: Юпітер, Діаметр: 139820 км, Супутників: 95
$planet4: Планета: Невідома, Діаметр: 0 км, Супутників: 0
```
