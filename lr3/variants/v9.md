# Варіант 9 — Лабораторна робота №3

> Варіант: 9
> Лаба: LR3 — ООП в PHP

## Завдання

### Завдання 1: Створення класів та об'єктів

- Клас `Artist` із властивостями: `name` (string), `style` (string), `exhibitions` (int)
- Створити 3 об'єкти з довільними значеннями:

| # | name | style | exhibitions |
|---|------|-------|-------------|
| 1 | Оксана Вовк | Імпресіонізм | 12 |
| 2 | Ігор Козлов | Абстракція | 8 |
| 3 | Тетяна Лебеденко | Реалізм | 15 |

### Завдання 2: Методи об'єкта

- Метод `getInfo()` — повертає рядок: `"Художник: {name}, Стиль: {style}, Виставки: {exhibitions}"`
- Викликати для кожного об'єкта

**Очікуваний результат:**
```
$artist1: Художник: Оксана Вовк, Стиль: Імпресіонізм, Виставки: 12
$artist2: Художник: Ігор Козлов, Стиль: Абстракція, Виставки: 8
$artist3: Художник: Тетяна Лебеденко, Стиль: Реалізм, Виставки: 15
```

### Завдання 3: Конструктор

- Конструктор задає початкові значення `name`, `style`, `exhibitions`
- Перестворити 3 об'єкти через конструктор (ті самі дані)

### Завдання 4: Клонування

- Метод `__clone()` — при копіюванні задає значення за замовчанням:
  - `name = "Художник"`, `style = ""`, `exhibitions = 0`
- Створити 4-й об'єкт через `clone` (від $artist3)

**Очікуваний результат:**
```
$artist3: Художник: Тетяна Лебеденко, Стиль: Реалізм, Виставки: 15
$artist4: Художник: Художник, Стиль: , Виставки: 0
```
