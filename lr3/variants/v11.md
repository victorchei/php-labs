# Варіант 11 — Лабораторна робота №3

> Варіант: 11
> Лаба: LR3 — ООП в PHP

## Завдання

### Завдання 1: Створення класів та об'єктів

- Клас `Book` із властивостями: `title` (string), `author` (string), `year` (int)
- Створити 3 об'єкти з довільними значеннями:

| # | title | author | year |
|---|-------|--------|------|
| 1 | Тіні забутих предків | Михайло Коцюбинський | 1911 |
| 2 | Кайдашева сім'я | Іван Нечуй-Левицький | 1879 |
| 3 | Захар Беркут | Іван Франко | 1883 |

### Завдання 2: Методи об'єкта

- Метод `getInfo()` — повертає рядок: `"Книга: {title}, Автор: {author}, Рік: {year}"`
- Викликати для кожного об'єкта

**Очікуваний результат:**
```
$book1: Книга: Тіні забутих предків, Автор: Михайло Коцюбинський, Рік: 1911
$book2: Книга: Кайдашева сім'я, Автор: Іван Нечуй-Левицький, Рік: 1879
$book3: Книга: Захар Беркут, Автор: Іван Франко, Рік: 1883
```

### Завдання 3: Конструктор

- Конструктор задає початкові значення `title`, `author`, `year`
- Перестворити 3 об'єкти через конструктор (ті самі дані)

### Завдання 4: Клонування

- Метод `__clone()` — при копіюванні задає значення за замовчанням:
  - `title = "Без назви"`, `author = ""`, `year = 0`
- Створити 4-й об'єкт через `clone` (від $book3)

**Очікуваний результат:**
```
$book3: Книга: Захар Беркут, Автор: Іван Франко, Рік: 1883
$book4: Книга: Без назви, Автор: , Рік: 0
```
