# Варіант 24 — Лабораторна робота №3

> Варіант: 24
> Лаба: LR3 — ООП в PHP

## Завдання

### Завдання 1: Створення класів та об'єктів

- Клас `Animal` із властивостями: `name` (string), `species` (string), `age` (int)
- Створити 3 об'єкти з довільними значеннями:

| # | name | species | age |
|---|------|---------|-----|
| 1 | Барсік | Кіт | 5 |
| 2 | Рекс | Собака | 3 |
| 3 | Кеша | Папуга | 8 |

### Завдання 2: Методи об'єкта

- Метод `getInfo()` — повертає рядок: `"Тварина: {name}, Вид: {species}, Вік: {age} р."`
- Викликати для кожного об'єкта

**Очікуваний результат:**
```
$animal1: Тварина: Барсік, Вид: Кіт, Вік: 5 р.
$animal2: Тварина: Рекс, Вид: Собака, Вік: 3 р.
$animal3: Тварина: Кеша, Вид: Папуга, Вік: 8 р.
```

### Завдання 3: Конструктор

- Конструктор задає початкові значення `name`, `species`, `age`
- Перестворити 3 об'єкти через конструктор (ті самі дані)

### Завдання 4: Клонування

- Метод `__clone()` — при копіюванні задає значення за замовчанням:
  - `name = "Без імені"`, `species = ""`, `age = 0`
- Створити 4-й об'єкт через `clone` (від $animal3)

**Очікуваний результат:**
```
$animal3: Тварина: Кеша, Вид: Папуга, Вік: 8 р.
$animal4: Тварина: Без імені, Вид: , Вік: 0 р.
```
