# Варіант 19 — Лабораторна робота №3

> Варіант: 19
> Лаба: LR3 — ООП в PHP

## Завдання

### Завдання 1: Створення класів та об'єктів

- Клас `Recipe` із властивостями: `name` (string), `cuisine` (string), `time` (int)
- Створити 3 об'єкти з довільними значеннями:

| # | name | cuisine | time |
|---|------|---------|------|
| 1 | Борщ | Українська | 120 |
| 2 | Вареники | Українська | 90 |
| 3 | Голубці | Українська | 150 |

### Завдання 2: Методи об'єкта

- Метод `getInfo()` — повертає рядок: `"Рецепт: {name}, Кухня: {cuisine}, Час: {time} хв"`
- Викликати для кожного об'єкта

**Очікуваний результат:**
```
$recipe1: Рецепт: Борщ, Кухня: Українська, Час: 120 хв
$recipe2: Рецепт: Вареники, Кухня: Українська, Час: 90 хв
$recipe3: Рецепт: Голубці, Кухня: Українська, Час: 150 хв
```

### Завдання 3: Конструктор

- Конструктор задає початкові значення `name`, `cuisine`, `time`
- Перестворити 3 об'єкти через конструктор (ті самі дані)

### Завдання 4: Клонування

- Метод `__clone()` — при копіюванні задає значення за замовчанням:
  - `name = "Новий рецепт"`, `cuisine = ""`, `time = 0`
- Створити 4-й об'єкт через `clone` (від $recipe3)

**Очікуваний результат:**
```
$recipe3: Рецепт: Голубці, Кухня: Українська, Час: 150 хв
$recipe4: Рецепт: Новий рецепт, Кухня: , Час: 0 хв
```
