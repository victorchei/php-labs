# Варіант 6 — Лабораторна робота №3

> Варіант: 6
> Лаба: LR3 — ООП в PHP

## Завдання

### Завдання 1: Створення класів та об'єктів

- Клас `Programmer` із властивостями: `name` (string), `language` (string), `level` (string)
- Створити 3 об'єкти з довільними значеннями:

| # | name | language | level |
|---|------|----------|-------|
| 1 | Віктор Юрченко | PHP | Senior |
| 2 | Катерина Ковальчук | JavaScript | Middle |
| 3 | Денис Хоменко | Python | Junior |

### Завдання 2: Методи об'єкта

- Метод `getInfo()` — повертає рядок: `"Програміст: {name}, Мова: {language}, Рівень: {level}"`
- Викликати для кожного об'єкта

**Очікуваний результат:**
```
$programmer1: Програміст: Віктор Юрченко, Мова: PHP, Рівень: Senior
$programmer2: Програміст: Катерина Ковальчук, Мова: JavaScript, Рівень: Middle
$programmer3: Програміст: Денис Хоменко, Мова: Python, Рівень: Junior
```

### Завдання 3: Конструктор

- Конструктор задає початкові значення `name`, `language`, `level`
- Перестворити 3 об'єкти через конструктор (ті самі дані)

### Завдання 4: Клонування

- Метод `__clone()` — при копіюванні задає значення за замовчанням:
  - `name = "Програміст"`, `language = ""`, `level = ""`
- Створити 4-й об'єкт через `clone` (від $programmer3)

**Очікуваний результат:**
```
$programmer3: Програміст: Денис Хоменко, Мова: Python, Рівень: Junior
$programmer4: Програміст: Програміст, Мова: , Рівень:
```
