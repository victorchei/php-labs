# Варіант 27 — Лабораторна робота №3

> Варіант: 27
> Лаба: LR3 — ООП в PHP

## Завдання

### Завдання 1: Створення класів та об'єктів

- Клас `University` із властивостями: `name` (string), `city` (string), `students` (int)
- Створити 3 об'єкти з довільними значеннями:

| # | name | city | students |
|---|------|------|----------|
| 1 | КПІ | Київ | 36000 |
| 2 | ЛНУ | Львів | 25000 |
| 3 | ХНУ | Харків | 18000 |

### Завдання 2: Методи об'єкта

- Метод `getInfo()` — повертає рядок: `"Університет: {name}, Місто: {city}, Студентів: {students}"`
- Викликати для кожного об'єкта

**Очікуваний результат:**
```
$university1: Університет: КПІ, Місто: Київ, Студентів: 36000
$university2: Університет: ЛНУ, Місто: Львів, Студентів: 25000
$university3: Університет: ХНУ, Місто: Харків, Студентів: 18000
```

### Завдання 3: Конструктор

- Конструктор задає початкові значення `name`, `city`, `students`
- Перестворити 3 об'єкти через конструктор (ті самі дані)

### Завдання 4: Клонування

- Метод `__clone()` — при копіюванні задає значення за замовчанням:
  - `name = ""`, `city = ""`, `students = 0`
- Створити 4-й об'єкт через `clone` (від $university3)

**Очікуваний результат:**
```
$university3: Університет: ХНУ, Місто: Харків, Студентів: 18000
$university4: Університет: , Місто: , Студентів: 0
```
