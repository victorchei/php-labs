# Варіант 21 — Лабораторна робота №3

> Варіант: 21
> Лаба: LR3 — ООП в PHP

## Завдання

### Завдання 1: Створення класів та об'єктів

- Клас `Employee` із властивостями: `name` (string), `department` (string), `salary` (float)
- Створити 3 об'єкти з довільними значеннями:

| # | name | department | salary |
|---|------|------------|--------|
| 1 | Анна Ващенко | IT-відділ | 55000.00 |
| 2 | Петро Гончаренко | Маркетинг | 38000.00 |
| 3 | Лариса Демченко | Фінанси | 47000.00 |

### Завдання 2: Методи об'єкта

- Метод `getInfo()` — повертає рядок: `"Працівник: {name}, Відділ: {department}, Зарплата: {salary} грн"`
- Викликати для кожного об'єкта

**Очікуваний результат:**
```
$employee1: Працівник: Анна Ващенко, Відділ: IT-відділ, Зарплата: 55000 грн
$employee2: Працівник: Петро Гончаренко, Відділ: Маркетинг, Зарплата: 38000 грн
$employee3: Працівник: Лариса Демченко, Відділ: Фінанси, Зарплата: 47000 грн
```

### Завдання 3: Конструктор

- Конструктор задає початкові значення `name`, `department`, `salary`
- Перестворити 3 об'єкти через конструктор (ті самі дані)

### Завдання 4: Клонування

- Метод `__clone()` — при копіюванні задає значення за замовчанням:
  - `name = "Працівник"`, `department = ""`, `salary = 0.0`
- Створити 4-й об'єкт через `clone` (від $employee3)

**Очікуваний результат:**
```
$employee3: Працівник: Лариса Демченко, Відділ: Фінанси, Зарплата: 47000 грн
$employee4: Працівник: Працівник, Відділ: , Зарплата: 0 грн
```
