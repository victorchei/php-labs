# Варіант 14 — Лабораторна робота №3

> Варіант: 14
> Лаба: LR3 — ООП в PHP

## Завдання

### Завдання 1: Створення класів та об'єктів

- Клас `Movie` із властивостями: `title` (string), `director` (string), `year` (int)
- Створити 3 об'єкти з довільними значеннями:

| # | title | director | year |
|---|-------|----------|------|
| 1 | Тіні забутих предків | Сергій Параджанов | 1965 |
| 2 | Плем'я | Мирослав Слабошпицький | 2014 |
| 3 | Атлантида | Валентин Васянович | 2019 |

### Завдання 2: Методи об'єкта

- Метод `getInfo()` — повертає рядок: `"Фільм: {title}, Режисер: {director}, Рік: {year}"`
- Викликати для кожного об'єкта

**Очікуваний результат:**
```
$movie1: Фільм: Тіні забутих предків, Режисер: Сергій Параджанов, Рік: 1965
$movie2: Фільм: Плем'я, Режисер: Мирослав Слабошпицький, Рік: 2014
$movie3: Фільм: Атлантида, Режисер: Валентин Васянович, Рік: 2019
```

### Завдання 3: Конструктор

- Конструктор задає початкові значення `title`, `director`, `year`
- Перестворити 3 об'єкти через конструктор (ті самі дані)

### Завдання 4: Клонування

- Метод `__clone()` — при копіюванні задає значення за замовчанням:
  - `title = "Без назви"`, `director = ""`, `year = 0`
- Створити 4-й об'єкт через `clone` (від $movie3)

**Очікуваний результат:**
```
$movie3: Фільм: Атлантида, Режисер: Валентин Васянович, Рік: 2019
$movie4: Фільм: Без назви, Режисер: , Рік: 0
```
