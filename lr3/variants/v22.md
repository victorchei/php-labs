# Варіант 22 — Лабораторна робота №3

> Варіант: 22
> Лаба: LR3 — ООП в PHP

## Завдання

### Завдання 1: Створення класів та об'єктів

- Клас `Course` із властивостями: `title` (string), `instructor` (string), `hours` (int)
- Створити 3 об'єкти з довільними значеннями:

| # | title | instructor | hours |
|---|-------|------------|-------|
| 1 | Web-розробка | Олександр Коваленко | 120 |
| 2 | Бази даних | Наталія Петренко | 80 |
| 3 | Алгоритми | Віктор Шевченко | 60 |

### Завдання 2: Методи об'єкта

- Метод `getInfo()` — повертає рядок: `"Курс: {title}, Викладач: {instructor}, Годин: {hours}"`
- Викликати для кожного об'єкта

**Очікуваний результат:**
```
$course1: Курс: Web-розробка, Викладач: Олександр Коваленко, Годин: 120
$course2: Курс: Бази даних, Викладач: Наталія Петренко, Годин: 80
$course3: Курс: Алгоритми, Викладач: Віктор Шевченко, Годин: 60
```

### Завдання 3: Конструктор

- Конструктор задає початкові значення `title`, `instructor`, `hours`
- Перестворити 3 об'єкти через конструктор (ті самі дані)

### Завдання 4: Клонування

- Метод `__clone()` — при копіюванні задає значення за замовчанням:
  - `title = "Новий курс"`, `instructor = ""`, `hours = 0`
- Створити 4-й об'єкт через `clone` (від $course3)

**Очікуваний результат:**
```
$course3: Курс: Алгоритми, Викладач: Віктор Шевченко, Годин: 60
$course4: Курс: Новий курс, Викладач: , Годин: 0
```
