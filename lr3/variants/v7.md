# Варіант 7 — Лабораторна робота №3

> Варіант: 7
> Лаба: LR3 — ООП в PHP

## Завдання

### Завдання 1: Створення класів та об'єктів

- Клас `Chef` із властивостями: `name` (string), `cuisine` (string), `stars` (int)
- Створити 3 об'єкти з довільними значеннями:

| # | name | cuisine | stars |
|---|------|---------|-------|
| 1 | Галина Поліщук | Українська | 3 |
| 2 | Роман Зайцев | Італійська | 2 |
| 3 | Юлія Сидоренко | Французька | 1 |

### Завдання 2: Методи об'єкта

- Метод `getInfo()` — повертає рядок: `"Шеф-кухар: {name}, Кухня: {cuisine}, Зірки: {stars}"`
- Викликати для кожного об'єкта

**Очікуваний результат:**
```
$chef1: Шеф-кухар: Галина Поліщук, Кухня: Українська, Зірки: 3
$chef2: Шеф-кухар: Роман Зайцев, Кухня: Італійська, Зірки: 2
$chef3: Шеф-кухар: Юлія Сидоренко, Кухня: Французька, Зірки: 1
```

### Завдання 3: Конструктор

- Конструктор задає початкові значення `name`, `cuisine`, `stars`
- Перестворити 3 об'єкти через конструктор (ті самі дані)

### Завдання 4: Клонування

- Метод `__clone()` — при копіюванні задає значення за замовчанням:
  - `name = "Шеф-кухар"`, `cuisine = ""`, `stars = 0`
- Створити 4-й об'єкт через `clone` (від $chef3)

**Очікуваний результат:**
```
$chef3: Шеф-кухар: Юлія Сидоренко, Кухня: Французька, Зірки: 1
$chef4: Шеф-кухар: Шеф-кухар, Кухня: , Зірки: 0
```
