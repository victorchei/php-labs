# Варіант 28 — Лабораторна робота №3

> Варіант: 28
> Лаба: LR3 — ООП в PHP

## Завдання

### Завдання 1: Створення класів та об'єктів

- Клас `Ticket` із властивостями: `event` (string), `venue` (string), `price` (float)
- Створити 3 об'єкти з довільними значеннями:

| # | event | venue | price |
|---|-------|-------|-------|
| 1 | Океан Ельзи | Палац спорту | 1200.00 |
| 2 | Вар'яти | Театр ім. Франка | 650.00 |
| 3 | Тінь Сонця | Stereo Plaza | 900.00 |

### Завдання 2: Методи об'єкта

- Метод `getInfo()` — повертає рядок: `"Квиток: {event}, Місце: {venue}, Ціна: {price} грн"`
- Викликати для кожного об'єкта

**Очікуваний результат:**
```
$ticket1: Квиток: Океан Ельзи, Місце: Палац спорту, Ціна: 1200 грн
$ticket2: Квиток: Вар'яти, Місце: Театр ім. Франка, Ціна: 650 грн
$ticket3: Квиток: Тінь Сонця, Місце: Stereo Plaza, Ціна: 900 грн
```

### Завдання 3: Конструктор

- Конструктор задає початкові значення `event`, `venue`, `price`
- Перестворити 3 об'єкти через конструктор (ті самі дані)

### Завдання 4: Клонування

- Метод `__clone()` — при копіюванні задає значення за замовчанням:
  - `event = "Новий квиток"`, `venue = ""`, `price = 0.0`
- Створити 4-й об'єкт через `clone` (від $ticket3)

**Очікуваний результат:**
```
$ticket3: Квиток: Тінь Сонця, Місце: Stereo Plaza, Ціна: 900 грн
$ticket4: Квиток: Новий квиток, Місце: , Ціна: 0 грн
```
