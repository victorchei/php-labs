# Варіант 5 — Лабораторна робота №3

> Варіант: 5
> Лаба: LR3 — ООП в PHP

## Завдання

### Завдання 1: Створення класів та об'єктів

- Клас `Musician` із властивостями: `name` (string), `instrument` (string), `yearsPlaying` (int)
- Створити 3 об'єкти з довільними значеннями:

| # | name | instrument | yearsPlaying |
|---|------|------------|--------------|
| 1 | Олексій Лисенко | Гітара | 10 |
| 2 | Марина Даниленко | Фортепіано | 7 |
| 3 | Артем Руденко | Скрипка | 14 |

### Завдання 2: Методи об'єкта

- Метод `getInfo()` — повертає рядок: `"Музикант: {name}, Інструмент: {instrument}, Стаж: {yearsPlaying} р."`
- Викликати для кожного об'єкта

**Очікуваний результат:**
```
$musician1: Музикант: Олексій Лисенко, Інструмент: Гітара, Стаж: 10 р.
$musician2: Музикант: Марина Даниленко, Інструмент: Фортепіано, Стаж: 7 р.
$musician3: Музикант: Артем Руденко, Інструмент: Скрипка, Стаж: 14 р.
```

### Завдання 3: Конструктор

- Конструктор задає початкові значення `name`, `instrument`, `yearsPlaying`
- Перестворити 3 об'єкти через конструктор (ті самі дані)

### Завдання 4: Клонування

- Метод `__clone()` — при копіюванні задає значення за замовчанням:
  - `name = "Музикант"`, `instrument = ""`, `yearsPlaying = 0`
- Створити 4-й об'єкт через `clone` (від $musician3)

**Очікуваний результат:**
```
$musician3: Музикант: Артем Руденко, Інструмент: Скрипка, Стаж: 14 р.
$musician4: Музикант: Музикант, Інструмент: , Стаж: 0 р.
```
