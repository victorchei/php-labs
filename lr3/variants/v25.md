# Варіант 25 — Лабораторна робота №3

> Варіант: 25
> Лаба: LR3 — ООП в PHP

## Завдання

### Завдання 1: Створення класів та об'єктів

- Клас `Country` із властивостями: `name` (string), `capital` (string), `population` (float)
- Створити 3 об'єкти з довільними значеннями:

| # | name | capital | population |
|---|------|---------|------------|
| 1 | Україна | Київ | 37.0 |
| 2 | Польща | Варшава | 38.0 |
| 3 | Чехія | Прага | 10.8 |

### Завдання 2: Методи об'єкта

- Метод `getInfo()` — повертає рядок: `"Країна: {name}, Столиця: {capital}, Населення: {population} млн"`
- Викликати для кожного об'єкта

**Очікуваний результат:**
```
$country1: Країна: Україна, Столиця: Київ, Населення: 37 млн
$country2: Країна: Польща, Столиця: Варшава, Населення: 38 млн
$country3: Країна: Чехія, Столиця: Прага, Населення: 10.8 млн
```

### Завдання 3: Конструктор

- Конструктор задає початкові значення `name`, `capital`, `population`
- Перестворити 3 об'єкти через конструктор (ті самі дані)

### Завдання 4: Клонування

- Метод `__clone()` — при копіюванні задає значення за замовчанням:
  - `name = ""`, `capital = ""`, `population = 0.0`
- Створити 4-й об'єкт через `clone` (від $country3)

**Очікуваний результат:**
```
$country3: Країна: Чехія, Столиця: Прага, Населення: 10.8 млн
$country4: Країна: , Столиця: , Населення: 0 млн
```
