# Варіант 1 — Лабораторна робота №3

> Варіант: 1
> Лаба: LR3 — ООП в PHP

## Завдання

### Завдання 1: Створення класів та об'єктів

- Клас `Student` із властивостями: `name` (string), `studentId` (string), `grade` (float)
- Створити 3 об'єкти з довільними значеннями:

| # | name | studentId | grade |
|---|------|-----------|-------|
| 1 | Іван Петренко | ST-001 | 4.5 |
| 2 | Олена Коваленко | ST-002 | 3.8 |
| 3 | Богдан Ткаченко | ST-003 | 4.2 |

### Завдання 2: Методи об'єкта

- Метод `getInfo()` — повертає рядок: `"Студент: {name}, ID: {studentId}, Середній бал: {grade}"`
- Викликати для кожного об'єкта

**Очікуваний результат:**
```
$student1: Студент: Іван Петренко, ID: ST-001, Середній бал: 4.5
$student2: Студент: Олена Коваленко, ID: ST-002, Середній бал: 3.8
$student3: Студент: Богдан Ткаченко, ID: ST-003, Середній бал: 4.2
```

### Завдання 3: Конструктор

- Конструктор задає початкові значення `name`, `studentId`, `grade`
- Перестворити 3 об'єкти через конструктор (ті самі дані)

### Завдання 4: Клонування

- Метод `__clone()` — при копіюванні задає значення за замовчанням:
  - `name = "Студент"`, `studentId = ""`, `grade = 0.0`
- Створити 4-й об'єкт через `clone` (від $student3)

**Очікуваний результат:**
```
$student3: Студент: Богдан Ткаченко, ID: ST-003, Середній бал: 4.2
$student4: Студент: Студент, ID: , Середній бал: 0
```
